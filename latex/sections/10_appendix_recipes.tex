\section{Crafting Recipes}
\label{app:recipes}

All twelve recipes use only vanilla Minecraft ingredients, with no dependency on any other
mod. Shapeless recipes (all but the power supply, function generator, and probe) are shown
as a loose set of ingredients; the three shaped recipes show their actual $3\times3$
crafting-grid layout. Every diagram was generated directly from the mod's own recipe
data files and the game's own item textures, so it is guaranteed to match what the game
itself accepts.
